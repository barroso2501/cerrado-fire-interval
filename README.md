## Overview

This repository contains the protocol, analysis code, and decision log for a study examining what fire frequency reveals — and conceals — about fire regimes in stable native vegetation of the Brazilian Cerrado over a 40-year period (1985–2024).

The central argument is that fire frequency, the most commonly used fire regime metric, is an aggregate measure that collapses temporal information. Two pixels with identical frequency may have radically different interval structures — with different implications for ecosystem function. This study characterizes that structure explicitly, including intervals that frequency ignores entirely: censored intervals and fire-free pixels.

---

## Research question

> Does fire frequency reliably indicate functional fire regimes in stable native vegetation of the Brazilian Cerrado, or do interval-level patterns reveal disruptions — by excess, exclusion, or both — that frequency conceals?

See `protocol/pre-registration.md` for the full pre-registered question, sub-questions, and analytical plan.

## Reproducibility

All analytical decisions are recorded in `protocol/decision-log.md` with date, rationale, and a flag indicating whether data had been seen at the time of the decision. Decisions made after first data access are explicitly marked **POST-HOC**.

Code is organized in numbered notebooks corresponding to analytical stages. Each notebook is committed to this repository at completion, with a commit message explaining what was done and why.

## Open science protocol

This study follows a pre-registration workflow to ensure that hypotheses, metrics, and threshold space are declared before any analytical access to the data. The workflow is:

1. Finalize `protocol/pre-registration.md`
2. Submit to OSF → receive DOI → add DOI to this README
3. Create Zenodo release v0.1 → freeze the protocol
4. Access data and begin analysis
5. Log every decision in `protocol/decision-log.md` as it is made
6. Commit notebooks and outputs at each milestone
7. Create Zenodo release v1.0 at manuscript submission
